Let's Flex!

This webpage is to allow anyone to easily experiment with flexbox container properties, and so will be able to better understand different container features of flexbox and how they work. First add items into the box and then start exploring using the dropdown options below, click "GO!" and see how the items are displayed on the page. The suns may go out of the box, what can you do to make them stay within the box?
